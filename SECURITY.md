# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in any Messy Virgo repository, please report it privately.

Preferred: Use GitHub "Report a vulnerability" (Private vulnerability reporting) on the affected repository.

Fallback: Email details to support@messyvirgo.com.

Please include:
- A description of the issue and potential impact
- Steps to reproduce (proof-of-concept if possible)
- Affected repository, branch/commit, and any relevant configuration

## Disclosure

Please do not open public GitHub issues for security vulnerabilities until a fix is available.

## Supported Versions

Only the latest version on the default branch is actively supported.
